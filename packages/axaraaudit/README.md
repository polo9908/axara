# axaraaudit

Alias exécutable de [`@axaraaudit/cli`](https://www.npmjs.com/package/@axaraaudit/cli) — audit design-system + RGAA 4.1 depuis le terminal.

```bash
npx axaraaudit audit
npx axaraaudit fix --write
npx axaraaudit voice src/Header.tsx
```
